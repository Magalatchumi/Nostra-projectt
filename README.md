# Nostra 
##E-Commerce Website

## Description
Nostra is a responsive e-commerce website developed using HTML, CSS, and JavaScript. It allows users to browse products, search collections, like products, and contact the store.

## Features
- Responsive Navigation Bar
- Mobile Sidebar Menu
- Offer Banner
- Hero Image Slider
- New Arrival Section
- Most Wanted Section
- Product Search
- Like/Unlike Feature
- Newsletter Subscription
- Contact Form

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome

## Project Files
- index.html
- collections.html
- contact.html
- style.css
- script.js

## Conclusion
This project demonstrates a modern responsive e-commerce website using front-end web technologies.
